# AIHelper — AI 提示词生成、优化与记忆管理

> 不再从零开始写提示词。让 AIHelper 帮你生成、优化并记住最好的提示词。

## 为什么需要 AIHelper？

与 AI（ChatGPT、Claude 等）对话时，提示词的质量直接决定输出的质量。但你是否有这些困扰：

- 用模糊的自然语言描述需求，结果总是不尽如人意
- 手动调整提示词很繁琐，改完就忘了哪个版本好用
- 优秀的提示词淹没在聊天记录里，再也找不到了

**AIHelper** 一次性解决这三个问题。将模糊需求转化为结构化提示词，一键优化现有提示词，并建立可搜索的记忆库。

## 功能

| 功能 | 优先级 | 说明 |
|------|--------|------|
| **提示词优化** | ⭐1 | 粘贴你的提示词 → AI 分析不足 → 输出改进版 |
| **提示词生成** | ⭐2 | 用自然语言描述需求 → 生成结构化提示词 |
| **模板库** | ⭐3 | 预置编程/写作/翻译/学习等模板，支持创建和分享 |
| **记忆管理** | ⭐4 | 标签分类、全文搜索、智能推荐历史提示词 |

## 工作原理

AIHelper 采用两阶段优化引擎：

1. **规则预处理** — 检测缺失元素（角色、上下文、格式、约束）并自动补全
2. **大模型精修** — 将预处理后的提示词发送给 AI 模型进行最终润色

结果是一个包含角色定义、任务描述、输出格式和质量约束的完整提示词——这些都是你平时容易忘记的。

## API Key

AIHelper **不提供** AI API。你需要自带任意支持商（OpenAI、Anthropic 等）的 API Key。Key 仅存储在本地。

## 使用方式

### 在线版

1. 访问 [oxelia51.com](https://oxelia51.com) 注册登录
2. 在工具菜单中打开 AIHelper
3. 在设置中填入你的 API Key
4. 开始优化提示词

### 桌面版（exe）

1. 从 [GitHub Releases](https://github.com/XiaoleC05/AIHelper/releases) 下载 `AIHelper.exe`
2. 运行即启动本地界面
3. 填入 API Key
4. 一切在本地运行，数据不上传
