# AIHelper — 提示词工坊

> 别再每次从头憋提示词了。让它帮你写、帮你改、帮你记住好用的那些。

## 痛点在哪

跟 AI 聊天时，话怎么说，决定了 AI 给你的答案好不好。但大部分人都有这几个烦恼：

- 脑子里的需求模模糊糊，跟 AI 说完它回的也不是你想要的
- 手动反复改提示词太累了，改完又不记得哪个版本好用
- 偶尔写出一条特别好的提示词，下次用时在聊天记录里翻半天找不着

**AIHelper** 把这三件事一次搞定。模糊想法变成精准提示词，旧提示词一键优化，还用过的提示词存一个能搜的仓库。

## 能做什么

| 功能 | 优先级 | 怎么用 |
|------|--------|--------|
| **优化提示词** | ⭐1 | 把写的不太好的提示词贴进来，它分析哪里不行，给你一个改好的 |
| **生成提示词** | ⭐2 | 用大白话说你要干啥，它帮你写成结构完整的提示词 |
| **模板市场** | ⭐3 | 编程、写文案、翻译、学习都有现成模板，自己也能建、能分享 |
| **记忆仓库** | ⭐4 | 用过的提示词打标签、搜全文、系统还会自动推荐你可能需要的 |

## 怎么优化的

分两步走：

1. **先补漏** — 检查你的提示词少了什么（比如没写角色、没写格式要求），自动帮你补上
2. **再润色** — 把补好的版本发给大模型，让它做最后的精修

最后出来的是一个角色明确、任务清楚、格式规范、质量要求到位的完整提示词。这些都是你平时容易漏掉的。

## 费用说明

AIHelper **不送** API。你需要用自己的 API Key（OpenAI、Claude 随便哪家都行）。Key 只存在你本地，平台不碰。

## 怎么用

### 线上

1. 打开 [oxelia51.com](https://oxelia51.com) 注册登录
2. 工具菜单里打开 AIHelper
3. 在设置里填上你的 API Key
4. 开始优化提示词

### 桌面版

1. 去 [GitHub Releases](https://github.com/XiaoleC05/AIHelper/releases) 下载 `AIHelper.exe`
2. 双击运行
3. 填上 API Key
4. 全部在本地跑，数据不往外传
